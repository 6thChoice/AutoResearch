# DLinear论文复现项目上下文

## 项目目标与愿景
- **核心目标**：完整复现论文《Are Transformers Effective for Time Series Forecasting?》（AAAI 2023 Oral, arXiv:2205.13504）中提出的DLinear模型实验与结果
- **学术价值**：验证DLinear作为时间序列预测简单有效基准方法的有效性，挑战Transformer在该领域的有效性假设，为后续研究提供可靠参考
- **工程交付**：产出可复现的代码库、实验记录与对比分析报告

## 技术栈与架构概述
- **核心框架**：PyTorch
- **数据处理**：pandas, numpy
- **实验管理**：自定义配置脚本（基于argparse或hydra）
- **评估指标**：MSE, MAE
- **模型架构**：DLinear（分解+线性层）
- **数据集**：ETT（电力变压器温度）、Weather、Electricity、Traffic
