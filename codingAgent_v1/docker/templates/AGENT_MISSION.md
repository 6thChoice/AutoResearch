## 快速参考

### 重要文件路径

- 配置文件: `config/settings.json`
- 工作目录: `volumes/workspace/`
- 日志目录: `volumes/logs/`

### 常用命令

```bash
# 查看容器状态
docker ps | grep claude

# 查看执行日志
tail -f volumes/logs/claude_runtime.log

# 进入容器
docker exec -it claude-code-agent bash
```
