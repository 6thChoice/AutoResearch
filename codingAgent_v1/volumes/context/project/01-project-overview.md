## 项目目标与愿景
本项目的核心目标是复现ICLR 2023论文《DLinear: Decomposition Linear Model for Long-term Time Series》中的实验工作。具体愿景是：通过完整复现训练和评估流程，独立验证DLinear模型在多个标准长期时间序列预测数据集（如ETT、Weather等）上的性能表现，确保获得与论文报告相近的结果（MSE、MAE等指标），以加深对模型原理的理解并建立可靠的代码基础。

## 技术栈与架构概述
*   **编程语言**： Python
*   **深度学习框架**： PyTorch
*   **核心模型架构**： DLinear模型，其核心思想是“先分解，后预测”。首先使用移动平均法将输入时间序列分解为趋势（Trend）和余项（Seasonal）分量。随后，使用两个独立的单层线性网络分别对两个分量进行预测，最后将预测结果相加得到最终输出。
*   **关键技术组件**： 标准时间序列数据加载与处理流程（标准化/归一化）、移动平均分解模块、线性层预测模块、基于均方误差（MSE）的模型训练与评估。

## 关键设计决策
1.  **遵循原始实现**： 优先严格参照论文作者的官方代码库（GitHub）进行复现，包括数据预处理方式、模型结构定义和超参数设置，这是保证结果可比性的关键。
2.  **移动平均分解**： 采用与论文一致的移动平均方法进行时间序列分解，而非更复杂的分解技术（如STL），这是DLinear模型设计简单性与有效性的核心。
3.  **线性预测器**： 对趋势和季节性分量分别使用简单的线性层进行映射，强调模型“线性”的本质，以证明其在长期预测任务上的竞争力。
4.  **标准化处理**： 训练前将对数据集按通道进行标准化（减去均值、除以标准差），并在预测后反标准化以计算真实尺度下的误差指标，这是时间序列预测的通用且必要步骤。
5.  **可复现性**： 将设置固定的随机种子（如 `seed=2021`），以确保多次运行的训练和评估结果一致，便于调试和结果验证。
6.  **任务驱动执行**： 使用 `AGENT_MISSION.md` 文档来明确指导实施步骤和跟踪进度，确保工作有序推进。
